# PROMPTS de imágenes (ejemplo)

Copia este archivo a `assets/imgs/<unidad>/PROMPTS.md` y escribe un prompt por
imagen. Genera el arte con **NanoBanana** (Gemini) vía Antigravity/Gemini CLI y
guarda cada salida con el nombre que espera el `\figura` del documento.
Ver `knowledge/imagenes-nanobanana.md`.

## Línea gráfica (constante para todo el tema)

- Plano y editorial (flat), tipo libro de texto.
- Color de la asignatura dominante + tints; ámbar solo como acento.
- SIN texto incrustado. Fondo limpio. Sin sombras pesadas.

## Imágenes

### `fracciones/pizza-mitades.png`  (relación 1:1)
```
Ilustración editorial plana de una pizza dividida en dos mitades iguales,
una mitad resaltada. Paleta azul (matemáticas) dominante + tints; acento
ámbar puntual. Composición centrada, sin texto, fondo claro.
```

### `fracciones/portada.png`  (relación 16:7, portada de sección)
```
Ilustración editorial plana sobre repartir en partes iguales (barras de
chocolate y vasos de jugo divididos). Paleta azul dominante. Sin texto.
```
